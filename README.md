# Gen AI Prompts Repository

This repository is a curated collection of advanced Generative AI prompts. These prompts are designed to instruct a Generative AI to act as a specific persona and perform a defined task, providing a structured and high-quality output.

## How to Use

Each markdown file in this repository represents a specific "prompt." To use a prompt, you need to:

1.  **Choose a prompt:** Browse the available prompts below and select the one that best suits your needs.
2.  **Open the prompt file:** Open the corresponding markdown file to view the prompt's content.
3.  **Copy the entire content of the file.**
4.  **Paste it into your favorite Generative AI tool.**
5.  **Follow the instructions within the prompt:** Many prompts require you to provide additional information, such as your resume, a job description, or a specific topic.
